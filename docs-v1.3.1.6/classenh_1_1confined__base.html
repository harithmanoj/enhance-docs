Detailed Description

template<class integral>
class enh::confined_base< integral >

Define an integral type that is confined to an interval.

The interval is specified by two functions set during construction. The interval check is done by other function set during construction.

template

integral : The integral type of the value.

Re-evaluates value according to predicate if anything changed.

Sets value to ulimit if exceeding upper, llimit if lower then lower limit.

This is used if the predicate changes according to other values,
