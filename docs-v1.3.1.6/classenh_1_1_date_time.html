◆ getStringDateTime() [2/2]

std::string enh::DateTime::getStringDateTime ( std::string  format)
inline

Get The date and time as a string in custom format.

Pass Argument as a string containing any of the following components.

Day : Name of day (Sunday) OR shDay : shortened name (Sun).
dd : The date (20) OR ddth : date with superscript (21st).
Month : The name of Month (January) OR mm : The numerical month (01 for January) OR shMonth : The name of month shortened (Jan).
yyyy : Year (2020).
sec : The seconds.
min : The minutes.
hour : The hours.
