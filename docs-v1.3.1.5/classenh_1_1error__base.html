Detailed Description

template<class type>
class enh::error_base< type >

The class for inheriting error tracking Functionality.

hasErrorHandlers = false;


Value 0x00 is reserved for Safe.
Value 0x01 is reserved for unknown errors.
Value 0x02 is reserved for invalid argument errors.

Template Argument

The underlying type to hold error information. (must be integral).
